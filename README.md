# 🚗 Hands on the Wheel Detection

A Computer Vision project that detects whether a driver's hands are on the steering wheel using the YOLO object detection model. This project aims to improve road safety by monitoring driver behavior in real time.

---

## 📌 Overview

Maintaining both hands on the steering wheel is essential for safe driving. This project uses a custom-trained YOLO model to detect the driver's hands and determine whether they are on the steering wheel.

The model can be used with:
- Images
- Videos
- Live webcam feed

---

## ✨ Features

- 🚗 Real-time driver monitoring
- ✋ Detects hands on the steering wheel
- 📷 Supports image, video, and webcam inference
- ⚡ Fast inference using YOLO
- 🎯 Custom-trained on a hands-on-wheel dataset

---

## 🛠️ Tech Stack

- Python
